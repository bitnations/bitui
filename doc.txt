BitUI Framework Documentation
===========================

Overview
--------
BitUI is a lightweight, flexible CSS framework focused on building modular interfaces with panels and a responsive grid system. It uses CSS variables for dynamic styling and supports both light and dark themes.

Core Concepts
------------

1. Layout Structure
   - Container > Row > Columns
   - Sections for grouping content
   - Panels as primary content blocks

2. Grid System
   - 12-column flexbox grid
   - Responsive breakpoints at 45em (720px) and 75em (960px)
   - Column classes: .col-1 through .col-12
   - Small screen variants: .col-1-sm through .col-12-sm
   - Rows use .row class with flex display

3. Panels
   - Basic panel: <div class="panel light">
   - Dark variant: <div class="panel dark">
   - Panels automatically handle spacing and borders
   - Used for grouping related content

4. CSS Variables
   --container-width: 90%
   --container-max-width: 65rem
   --border-radius: 15px
   --spacing-unit: 20px
   --panel-padding-x: 15px
   --panel-padding-y: 5px
   --font-size-base: 16px
   --line-height-base: 1.4

5. Typography
   - Base font: Roboto Mono (monospace fallback)
   - Headings: h1 through h4
   - Utility classes:
     .font-light   (300 weight)
     .font-regular (400 weight)
     .font-heavy   (700 weight)
     .font-small   (0.75x base)
     .font-large   (1.25x base)

6. Buttons
   - Basic: <button class="btn">
   - Colors: .green, .blue, .red, .purple
   - Disabled state: .disabled

Basic Template
-------------
<main class="content">
  <div class="container">
    <section>
      <div class="panel light">
        <div class="row">
          <div class="col-12">
            <h1>Title</h1>
          </div>
        </div>
      </div>
    </section>
  </div>
</main>

Common Patterns
--------------
1. Two-Column Layout:
   <div class="row">
     <div class="col-6">
       <div class="panel light">
         Left content
       </div>
     </div>
     <div class="col-6">
       <div class="panel light">
         Right content
       </div>
     </div>
   </div>

2. Panel Layout:
   <div class="panel light">
     <h3>Panel Title</h3>
     <p>Panel content</p>
     <button class="btn blue">Action</button>
   </div>

3. Form Group:
   <div class="panel light">
     <div class="row">
       <div class="col-12">
         <label>Label</label>
         <input type="text">
       </div>
     </div>
   </div>

Best Practices
-------------
1. Always wrap content in panels for consistent spacing
2. Use sections to group related panels
3. Maintain proper nesting: container > row > col > panel
4. Use utility classes for text alignment and styling
5. Keep panel content focused and modular
