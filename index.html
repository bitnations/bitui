bitUI

bitUI is a simple, lightweight HTML + CSS framework for building clean, responsive layouts. It features a flexible 12-column grid, utility classes, and minimal styling.

  • Adjust the layout with the sliders on the right
  • Download the CSS
  • Drop it into your site

The Grid

bitUI is a 12-column system using flexbox. col-1 to col-12. Columns go inside rows, and panels go inside columns. You can nest as many row, col, and panel elements as needed.

Layout

bitUI provides utility classes to align rows, columns, and text. row alignment affects columns, while col alignment affects text inside.

Typography

bitUI uses a clean, modern typography system based on the var(--font-family-base). All text adapts dynamically to our theme system.

Paragraph text is styled with a default font-weight of 400. You can modify font weights using utility classes:

.f-light

.f-regular

.f-heavy

Apply color utilities to text for thematic emphasis:

.f-danger – Red text

.f-confirm – Green text

.f-info – Blue text

.f-warning – Purple text

.f-white – White text

.f-black – Black text

.f-gray – Gray text
