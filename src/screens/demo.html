BitUI

BitUI is a free-to-use, lightweight HTML + CSS frontend framework designed for simplicity and performance. Built as the foundation for BitNations, BitUI provides a minimal, flexible grid system and utility classes to help you build responsive, fast-loading web applications with ease.
