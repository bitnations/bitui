BitUI

BitUI is a free-to-use, lightweight HTML + CSS frontend framework designed for simplicity and performance. Built as the foundation for BitNations, BitUI provides a minimal, flexible grid system to build responsive, fast-loading web applications with ease.

Typography

BitUI uses a clean, modern typography system based on the var(--font-family-base) font stack (Roboto Mono, NotoColorEmojiLimited, monospace). All text adapts dynamically to our theme system (Light, Warm, Dim, Dark) using global variables like --text-color and --heading-color.

Paragraph text is styled with a default font-weight of 400. You can modify font weights using utility classes:

.font-light

.font-regular

.font-heavy

Apply color utilities to text for thematic emphasis:

.f-black – Black text

.f-white – White text

.f-red – Red text

.f-orange – Orange text

.f-green – Green text

.f-blue – Blue text

.f-purple – Purple text

.f-gray – Gray text
